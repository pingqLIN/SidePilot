SDK 模式登入引導

首次使用 SDK 模式需要先登入 GitHub Copilot 帳號

  1. 確認 `scripts/copilot-bridge` 已啟動
  2. 開啟 GitHub 後完成登入
  3. 回到 SidePilot 開始聊天

SidePilot

在側邊欄直接使用 GitHub Copilot

  • 🖥️ 側邊面板固定顯示，瀏覽網頁不中斷
  • 📋 一鍵擷取頁面內容，提供給 Copilot 參考
  • 💻 支援程式碼區塊、標題結構擷取
⚡ 使用方式

透過內嵌框架直接連接 GitHub Copilot，需使用自己的 GitHub 帳號登入。所有對話直接與 GitHub 進行，不涉及 API 代理或中間伺服器。

🔒 所有資料僅在本地處理，不會上傳至第三方
⚠️ 使用風險與免責聲明
  • 本擴充功能繞過安全標頭嵌入 Copilot，可能違反 GitHub 服務條款
  • 非 GitHub 官方授權，使用可能導致帳號被暫停或終止
  • 需擁有有效的 GitHub Copilot 訂閱
  • 開發者不承擔任何因使用而導致的損害

要啟用 iframe 模式嗎？

把 GitHub Copilot 網頁直接嵌入側邊欄

  • ✅免安裝、免設定，開啟即用
  • ⚠️需移除 GitHub 安全標頭，屬 ToS 灰色地帶
  • 🔧偏好進階功能（記憶庫、規則、串流）請選「否」使用 SDK 模式

隨時可在 設定 › iframe 模式 中修改此選項
