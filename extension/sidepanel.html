歡迎使用 SidePilot

  • ✅ 側邊面板固定顯示，瀏覽網頁不中斷
  • ✅ 一鍵擷取頁面內容，提供給 Copilot 參考
  • ✅ 支援程式碼區塊、標題結構擷取
⚡ 使用方式：

本擴充功能透過內嵌框架顯示 GitHub Copilot 官方網頁，您需要使用自己的 GitHub 帳號登入。所有對話直接與 GitHub 進行，本擴充功能不涉及 API 代理或中間伺服器。

🔒 隱私說明：

頁面擷取功能僅在本地處理資料，不會上傳任何內容到第三方伺服器。

⚠️ 重要法律聲明：

本擴充功能處於法律灰色地帶，使用前請詳閱：

1. 安全標頭繞過

本擴充功能透過移除 X-Frame-Options 和 Content-Security-Policy 標頭來將 GitHub Copilot 嵌入 iframe。此技術繞過了 GitHub 設定的安全機制，可能違反 GitHub 服務條款第 H 條（API 條款）中關於濫用或過度請求的規定。

2. 非官方整合

SidePilot 並非 GitHub 官方授權或認可的擴充功能。GitHub Copilot 服務條款明確規定使用者應透過官方 IDE 擴充功能或 GitHub.com 存取服務。以非預期方式嵌入可能被視為違反使用條款。

3. 帳號風險

使用本擴充功能可能導致您的 GitHub 帳號被暫停或終止。根據 GitHub 服務條款第 L 條，GitHub 保留在任何時候以任何理由終止帳號的權利，包括違反服務條款的行為。

4. 訂閱條款

您必須擁有有效的 GitHub Copilot 訂閱才能使用此功能。本擴充功能不繞過付費機制，但以非官方方式存取付費服務仍可能違反訂閱條款。

5. 免責聲明

本擴充功能的開發者不承擔因使用本軟體而導致的任何直接、間接、附帶或後果性損害，包括但不限於帳號終止、訂閱取消或法律訴訟。

  • 理解並接受上述風險
  • 確認擁有有效的 GitHub Copilot 訂閱
  • 同意自行承擔所有法律與帳號安全責任
