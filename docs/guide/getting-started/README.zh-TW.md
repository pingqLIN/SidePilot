## 5 分鐘上手路線

1. clone 專案並安裝依賴
2. 把 `extension/` 載入成 Chrome 未封裝擴充
3. 從工具列打開 SidePilot
4. 先用 **iframe 模式** 快速開始
5. 想要串流、規則、記憶注入時再切到 **SDK 模式**
