## 5-minute route

1. Clone the repo and install dependencies
2. Load `extension/` as an unpacked Chrome extension
3. Open SidePilot from the toolbar
4. Start with **iframe mode** for instant use
5. Switch to **SDK mode** when you want streaming, rules, and memory injection
