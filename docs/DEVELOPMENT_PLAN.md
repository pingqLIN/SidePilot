# SidePilot 開發計畫書

## 專案名稱
SidePilot - Chrome 側邊欄 AI 助手

## 版本目標
v2.0 - 雙模式架構

---

## 目標

### 主要目標
提供「官方 SDK 模式」與「網頁嵌入模式」雙軌切換，提升專案合法合規性，同時保留便利性。

### 具體目標

| # | 目標 | 說明 |
|---|------|------|
| 1 | **合規化** | 透過官方 Copilot SDK 提供合法存取管道 |
| 2 | **雙模式切換** | 用戶可自由選擇 SDK 或 iframe 模式 |
| 3 | **視覺一致性** | 兩種模式維持相同外觀與操作體驗 |
| 4 | **向下相容** | 保留現有 iframe 模式供快速使用 |

---

## 模式比較

| 面向 | 🔒 SDK 模式 | ⚡ iframe 模式 |
|------|------------|---------------|
| 法律地位 | ✅ 官方支援 | ⚠️ 灰色地帶 |
| 帳號風險 | ✅ 無 | ⚠️ 有風險 |
| 前置需求 | Copilot CLI | 無 |
| 適用場景 | 長期使用 | 快速體驗 |
